# TrideUSDT Telegram Bot

Um bot de rendimento em USDT com retorno de 20% ao ano (APY), com botões interativos.

### Comandos

- `/start` — ativa o bot
- Mensagem qualquer — exibe o menu

### Variáveis no `env.json`

```json
{
  "BOT_TOKEN": "...",
  "ADMIN_ID": 123456789,
  "CARTEIRA_USDT": "..."
}
